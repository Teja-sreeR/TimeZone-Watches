# TimeZone Watches - E-Commerce Website

A simple and attractive e-commerce website for a watch company, created for educational purposes.

## Project Overview

TimeZone Watches is a single-page e-commerce website featuring:
- Home page with hero section
- Product catalog with 3 sample watches
- Order list and wishlist functionality
- Customer reviews and comments section
- About Us information
- Contact page with customer care details
- Report issue form
- User registration and login system
